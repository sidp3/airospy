\# 🔥 AI Fire \& Unknown Object Detection System



A real-time AI surveillance system using YOLOv8 and Computer Vision to detect:

\- 🔥 Fire hazards

\- ❓ Unknown / suspicious objects

\- 🎥 Live webcam, RTSP, and video input



---



\## 🚀 Features

\- Real-time YOLOv8 object detection

\- Fire detection with flicker analysis

\- Unknown object detection via motion analysis

\- Streamlit-based interactive UI

\- WebRTC webcam streaming



---



\## 🛠️ Tech Stack

\- Python

\- Streamlit

\- YOLOv8 (Ultralytics)

\- OpenCV

\- WebRTC
